# Surveillance Pricing — Jurisdiction Tracker

Interactive web app for navigating comparative regulation of surveillance pricing across 15 jurisdictions.

Interactive comparative regulation tracker for surveillance pricing research.

## Views

- **Topology Grid** — Baldwin's regulatory strategies mapped across 15 jurisdictions
- **Jurisdiction Detail** — Full molecule content for each jurisdiction
- **Strategy Filter** — Cross-jurisdiction view by strategy type
- **Cross-Cutting Analysis** — Emerging patterns, findings, and theoretical framework

## Data

- 15 jurisdiction molecules with full regulatory profiles
- ~150 instruments across 7 strategy types
- Structured data for filtering and navigation
- Full markdown content for deep reading

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

Push to GitHub, connect to [Streamlit Community Cloud](https://streamlit.io/cloud).
